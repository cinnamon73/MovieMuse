Movie Picker Icon Generator

This tool creates placeholder icons for the Movie Picker app.

Instructions:

  1. Click "Generate Icons" to create the placeholder icons
  2. Right-click each canvas and "Save image as..." OR use the download buttons
  3. Save the 1024x1024 image as app_icon.png in assets/icons/
  4. Save the 512x512 image as splash_icon.png in assets/icons/
  5. Run the Flutter commands to generate platform-specific icons

Flutter Commands:

flutter pub get
flutter pub run flutter_launcher_icons:main
flutter pub run flutter_native_splash:create
